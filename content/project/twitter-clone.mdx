This is a Twitter clone built with Next.js and OpenAI's GPT-3 API. This project is a great example of how to use Next.js to build a full-stack application with serverless functions.

## Features

- User authentication
- Create and delete tweets
- Like and retweet tweets
- Follow and unfollow users
- Real-time updates with WebSockets

## Technologies

- Next.js
- MongoDB
- OpenAI's GPT-3 API
- Vercel

## Installation

To run this project locally, you'll need to have Node.js and MongoDB installed. Then, you can clone the repository and install the dependencies:

```bash
git clone
cd twitter-clone
npm install
```

You'll also need to create a `.env.local` file with the following environment variables:

```bash
MONGODB_URI=your-mongodb-uri
JWT_SECRET=your-jwt-secret
GPT3_API_KEY=your-gpt3-api-key
```

Finally, you can start the development server:

```bash
npm run dev
```

## Deployment

This project is set up to be deployed to Vercel. You can deploy it by running the following command:

```bash
vercel
```
